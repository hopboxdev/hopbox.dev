Hopbox

Self-hosted SSH gateway that drops you into isolated Docker dev containers. No VPN, no browser, no IDE tunnel — just ssh.

$ ssh -p 2222 hop@hopbox.dev

Per-user boxes

Each SSH key gets its own Docker container with a persistent home directory.

Tool wizard

Pick your multiplexer, editor, shell, and runtimes on first connect.

Multi-device

Link new keys to existing boxes with a one-time code.

Observable

Structured logs, Prometheus metrics, Grafana dashboards.
